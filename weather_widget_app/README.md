# Weather Widget

## Overview

Weather Widget is a sleek and interactive weather application that provides real-time weather information. It features a user-friendly interface and is designed to offer an engaging user experience with dynamic weather updates.

## Live Demo

Check out the live demo [here](https://weatherwidget-blush.vercel.app/).

## Features

- **Real-time Weather Updates**: Get current weather information for any location.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Loading Animation**: Engaging loading animation while fetching data.

## Installation

To get started with the Weather Widget locally, follow these steps:

1. **Clone the Repository**

   ```bash
   git clone https://github.com/MuhammadAriyan/30-DAYS-30-PROJECTS.git
   cd 30-DAYS-30-PROJECTS/02_weather_widget_app
   ```

2. **Install Dependencies**

   Make sure you have [Node.js](https://nodejs.org/) installed. Then run:

   ```bash
   npm install
   ```

3. **Run the Development Server**

   ```bash
   npm run dev
   ```

   Open your browser and navigate to `http://localhost:3000` to view the widget.

## Usage

1. **Configure API Key**: Add your weather API key to the `.env.local` file:

   ```env
   WEATHER_API_KEY=your_api_key_here
   ```

2. **Modify Components**: Edit the components in the `src/components` folder to customize the widget according to your needs.

## Technologies

- **Next.js**: React framework for server-side rendering.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **ShadUI**: A UI library for building accessible and customizable components in React.
- **TypeScript**: Typed superset of JavaScript for a better development experience.
- **Vercel**: Deployment platform for hosting the widget.
